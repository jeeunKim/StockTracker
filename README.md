 # 지도 위의 떨이 마켓   *StockTracker* 
   ## 개발기간 
   ### 23.09 ~ 23.11 

   <br>

   ## 인원
   ### BE-2, FE-2 4인 팀프로젝트

  <br>
  
   ## 주요기능
    - 카카오, 네이버 소셜 로그인
    - 아이디, 비밀번호 찾기 휴대폰 인증 문자 발송
    - 회원가입 비밀번호 암호화 및 Validation
    - 페이지별 Intetceptor
    - 현재 위치를 기반으로 한 지도와 가게 마커표시
    - 거리, 마감기한, 가격, 별점 필터링 기능
    - 가게, 아이템, 공지사항, 문의 CRUD 및 Validation
    - 즐겨찾기한 가게, 공지사항 알림 기능
    - Scheduled를 이용한 아이템 자동삭제
    - 관리자 페이지
